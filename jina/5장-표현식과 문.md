## 5.6 표현식인 문과 표현식이 아닌 문
```javascript
var x; // 변수 선언문은 값으로 평가 될 수 없음
x= 1+2; // 표현식이면서 완전한 문
```

표현식 문 vs 표현식이 아닌 문
* 표현식 문: 값으로 평가되므로 변수에 할당 가능
* 표현식이 아닌 문: 변수에 할당 불가
